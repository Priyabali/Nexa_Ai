**Nexa** is a web-based AI assistant built using **HTML, CSS, and JavaScript**. It can greet you, tell the time, open websites, and search the internet using voice commands.

Check it out here: [Live Demo](https://nexa-yourassistant.netlify.app/)

---

## Features

* Voice recognition for commands
* Text-to-speech responses
* Time-based greetings
* Open websites like YouTube, Google, Facebook, Instagram, WhatsApp
* Google search for any query

---

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Priyabali/nexa.git
   ```
2. Open `index.html` in your browser.
3. Allow microphone access for voice commands.
